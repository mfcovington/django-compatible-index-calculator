The Amaryllis sicTool (Sequencing Index Compatibility Tool) helps prevent the accidental use of adapters with incompatible index sequences when performing Next Generation Sequencing experiments. Using the Amaryllis sicTool, researchers can easily and confidently combine libraries made with adapters even from unrelated index sets.


The Amaryllis sicTool has three modes. Use Interactive Mode to explore and visualize compatibility between index sequences from commercially available index sets. Automatic Mode is useful when you are busy preparing to synthesize libraries and want someone else to choose a good set of compatible index sequences for you. If you already have a set of index sequences, use Custom Mode to test their compatibility. All three modes allow you to download a sample sheet template populated with your chosen set of compatible index sequences.
